## System

SENTINEL runs as a desktop overlay during live sessions:

1. **Status pill** — always-on-top clock so the agent stays present without owning the screen.
2. **Capture panel** — thought entry, emotion tags, and optional lessons in one pass.
3. **Caution alert** — when the pattern matches, name it, explain the trap, and prescribe a short reset.

## Result

SENTINEL turns post-session journaling into an in-session interrupt. The design priority is speed and specificity: catch the urge, name it, and create a pause before the trade.
